# Add GGCHAIN to MetaMask

## Method 1: One-Click (Recommended)

1. Visit [gghyper.net](https://gghyper.net)
2. Scroll to "Network Information" section
3. Click **"Add Network"** button
4. MetaMask popup → **Approve**

## Method 2: Manual Setup

1. Open MetaMask → Settings → Networks → **Add Network**
2. Fill in:

| Field                | Value                          |
|----------------------|--------------------------------|
| Network Name         | GGCHAIN Mainnet                |
| New RPC URL          | `https://rpc.gghyper.net`      |
| Chain ID             | `2121217`                      |
| Currency Symbol      | `GG`                           |
| Block Explorer URL   | `https://explorer.gghyper.net` |

3. Click **Save**

## Verify Connection

After adding, your MetaMask balance will show in **GG**. You can:
- Send/receive GG natively
- View transactions on https://explorer.gghyper.net
- Connect to dApps deployed on GGCHAIN

## Add USDT Token

To see Tether GGPAY (USDT) balance:

1. MetaMask → **Tokens** tab → **Import Tokens**
2. Paste contract address: `0x79Cce540c444AE1D7bDfd318CfBDB950147BcEc0`
3. Symbol: `USDT`, Decimals: `18`
4. **Import**
